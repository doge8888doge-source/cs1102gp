What is Synthetic Identity Fraud?

  • A fabricated identity using mixed real + fake data (e.g., real SSN + fake name + random address)
  • Fraudsters build credit history over time (“piggybacking”) then default on loans
  • Hard to detect: no obvious single red flag → rules-based systems fail

Industry Impact

  • Responsible for billions in annual losses for banks, fintechs, and BNPL lenders
  • Traditional systems rely on static rules (e.g., duplicate phone, invalid zipcode) and miss relational fraud patterns
    • Key challenges:
    • Hidden connections between applications, devices, and identities
    • Evolving fraud tactics (deepfakes, synthetic data generation)
    • Balancing fraud detection with false positives (rejecting legitimate applicants)
    • Compliance with fair lending laws (anti-discrimination, transparency)
